# 心灵伙伴 - 学生心理陪伴系统

🫂 面向小学生的AI心理陪伴对话系统，支持语音交互、危机检测、心理档案管理。

---

## 一键启动（Windows/Mac/Linux）

### 方式1：直接运行EXE（推荐）
1. 下载 `心灵伙伴.exe`
2. 双击运行
3. 打开浏览器访问显示的地址

### 方式2：Python环境运行
```bash
# 安装依赖
pip install fastapi uvicorn requests python-dotenv

# 运行
python start.py
```

## 常见问题

### Q: 启动后显示"端口被占用"？
A: 程序会自动尝试其他端口，如 8001、8002 等。访问显示的地址即可。

### Q: 提示"数据库初始化失败"？
A: 确保程序所在目录有写入权限，或尝试以管理员身份运行。

### Q: AI对话显示"网络错误"？
A: 检查 `.env` 文件中的 `MINIMAX_API_KEY` 是否正确配置。

### Q: 管理端打不开？
A: 确保完整解压zip包（包含 `admin` 文件夹），访问 http://localhost:8000/admin/

### Q: 端口是多少？
A: 默认 8000，如被占用会自动切换。
